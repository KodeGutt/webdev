Media Typer

Media typer lar deg spesifisere CSS til forskjellige bruksområder. Hovedtypene er all, print, screen, og speech som lar deg endre utseende til en nettside. Ved å lage et nytt CSS ark eller en seksjon i koden din så kan du endre hvordan du vil at datamaskinen skal vise siden din. Screen er det som oftest blir brukt og velger et CSS ark som fungerer på alle skjermer. Det er vanlig å gjøre at nettsiden automatisk fungerer på alle skjermer, så gamle media typer som har egen CSS for TV og Prosjektorer er sjeldent brukt. Print er smart for å gjøre det lettere å skrive ut en nettside, ved å f.eks endre utseende til linker, navigasjon og header. Du kan også endre hvordan text to speech fungerer. Vanligvis kan du holde deg til Screen, men Print er kjekt hvis noen trenger en fysisk versjon av siden. Hvis du ikke gidder, kan du bare velge medietype All og ha ett enkelt CSS ark for alle tilfeller.
