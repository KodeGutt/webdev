Git/Github

Github er en cloud service som lar deg lagre og redigere kode på nett. Det som er spesielt med github er at det har versjonskontroll og verktøy som gjør det lett å samarbeide med andre utviklere. Når du skal endre noe på Github «Committer» du en ny versjon av filene du jobber med. Versjonskontroll gjør at tidligere versjoner forblir lagret og kan hentes ved behov. Andre brukere kan også sende en Commit Request hvor du kan velge å akseptere endringene de har gjort. Det er også mulig å rapportere bugs og problemer i koden.

Vanligvis er kode på Github Open Source, det vil si at hvem som helst kan kopiere og utvikle sin egen versjon av en kode. Dette kalles en Fork, og gjør det mulig for utviklere å tilpasse kode til sine egne systemer. Jeg synes Github er en fantastisk måte å lære på ved å se på kode lagd av profesjonelle. Det kan gi tips om struktur, koding og måter du kan simplifisere oppgaver.
